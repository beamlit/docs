Callout: Beamlit also lets you replicate a model deployment from your private Kubernetes cluster. To get started with this method, read this tutorial.

1. Introduction: what is Beamlit in 2 sentences, what this tutorial will let you do, + callout
2. Install beamlit CLI
3. Run beamlit login (with a private key)
4. Run beamlit serve with a first model
5. Make an inference with Beamlit CLI or by API
